Linked List with python

  • A linked list is a linear data structure where elements (nodes) are not stored contiguously in memory.
  • Each node contains data and a reference (pointer) to the next node, creating a chain-like structure.
  • Linked lists are ideal for dynamic data (frequently added or removed elements) due to efficient insertion and deletion operations, as opposed to arrays/lists that require shifting elements. 


 class Node:
     def __init__(self, data):
          self.data = data
          self.next = None
            
 class Blog:
     def __init__(self):
         self.head = None  # Head node, initially None

    def add_post(self, title, content):
        # Create a new node
        new_post = Node({"title": title, "content": content})

        # If the list is empty, set the new node as the head
        if self.head is None:
            self.head = new_post
            return

        # Traverse to the last node
        current = self.head
        while current.next:
            current = current.next

        # Append the new node to the end
        current.next = new_post

    def print_posts(self):
        current = self.head
        while current:
            print(f"Title: {current.data['title']}")
            print(f"Content: {current.data['content']}")
            print("-" * 20)
            current = current.next
